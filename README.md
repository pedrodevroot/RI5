# Projeto Empresa - TypeScript

Projeto desenvolvido para a disciplina de **Técnicas de Programação** da **FATEC**.

O objetivo do projeto é representar uma empresa utilizando **Programação Orientada a Objetos (POO)** em **TypeScript**, baseado em um **diagrama UML**.

---

## ⚙️ Como executar o projeto

1️⃣ Compilar o TypeScript:

```
tsc
```

2️⃣ Executar o arquivo compilado:

```
node dist/index.js
```
